# Attribution

This project includes code, display design, environmental value presentation and public-facing
documentation that require clear attribution.

## Required Public Attribution

Every public page, screen or display that shows measurements from the Technology Club of Thrace
Environmental Station must include:

- the Technology Club of Thrace logo;
- a clear "Measurement Source" label or equivalent;
- a link or reference to https://steth.gr/env.

The attribution must be visible to viewers and must not be hidden in source code only.

## Required Source and Documentation Attribution

Copies, forks and modified versions must preserve the following attribution in source code and
documentation:

```text
Code curation and educational adaptation:
Dimitrios Kanatas
https://labschool.gr
https://labschool.mysch.gr

Environmental data source:
Technology Club of Thrace Environmental Station
https://steth.gr/env
```

## Optional Local Display Attribution

Organizations that deploy the application may add a short local display notice, such as:

```text
Displayed by: [Organization Name]
```

This local notice must not replace, hide or weaken the original measurement source attribution.

## Suggested Citation

```text
Environmental Station Measurements Display Application.
Code curation and educational adaptation by Dimitrios Kanatas.
Environmental data from the Technology Club of Thrace Environmental Station.
https://steth.gr/env
```
