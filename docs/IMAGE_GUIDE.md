# Image Guide for GitHub Pages

The documentation site has two language versions:

- Greek default page: `docs/index.html`
- English page: `docs/en.html`

Place documentation images in:

```text
docs/assets/images/
```

Recommended image format:

- `.webp` for screenshots and background images;
- `.jpg` for real photos;
- width around 1600 px for hero/dashboard screenshots;
- width around 1200 px for smaller documentation images.
