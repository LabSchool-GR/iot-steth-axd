# GitHub Pages Documentation

This folder contains the static documentation site for the Environmental Station Measurements
Display Application.

Recommended GitHub Pages setting:

```text
Source: Deploy from a branch
Branch: main
Folder: /docs
```

Main files:

- `index.html` - Greek default documentation landing page for the display application;
- `en.html` - English documentation landing page;
- `styles.css` - sea-themed documentation styling;
- `IMAGE_GUIDE.md` - requested screenshots/photos and suggested filenames;
- `assets/images/` - folder for documentation images.

When screenshots are ready, place them in `docs/assets/images/` and replace the visual placeholders
in `index.html` with real `<img>` elements.
