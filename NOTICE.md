# Notice

Environmental Station Measurements Display Application

This project displays and documents environmental measurement values retrieved from the Technology
Club of Thrace Environmental Station through a web display and TV/kiosk presentation mode.

## Code Curation and Educational Adaptation

- Dimitrios Kanatas
- https://labschool.gr
- https://labschool.mysch.gr

## Environmental Data Source

- Technology Club of Thrace Environmental Station
- https://steth.gr/env

## Interface Assets

The measurement icons used in TV Mode are local inline SVG interface elements.
They do not require an external icon font, CDN or third-party runtime request.

## Attribution Requirement

Any public display, deployment, fork or derivative that presents measurements from the station must
keep visible attribution to the Technology Club of Thrace Environmental Station.

The Technology Club of Thrace logo, measurement source label and station link must not be removed
from public displays that use the station measurements.
